# 需求⽂档

# （1）什么是⽤⼾痛点？

• 定义要求：指⽤⼾在⾃考/学历提升咨询过程中，表现出的顾虑、担忧或阻碍其签约助学服务的核⼼卡点，核⼼是⽤⼾报名的核⼼决策障碍。

• ⽂字描述例⼦：痛点包括但不限于⽤⼾担⼼基础差跟不上学习、拿证周期太⻓、担⼼学历不被国家认可、害怕考试难度⼤、不清楚报考流程、担⼼1858元服务费⽤不值、顾虑后续学费是否有隐形消费、担⼼服务没有保障等。

# . 聊天样本例⼦：样本1：

• 学⽣：⽼师，我毕业好多年了，英语数学全忘光了，基础特别差，⾃考能跟上学习吗？⽼师：……学⽣：我平时要上班，只有晚上1个⼩时空闲，能跟得上学习进度吗？

# • 样本 2：

# （2）什么是成交节点？

• 定义要求：指破冰沟通后，⽼师清晰讲解1858元助学服务的全部权益、适配⽤⼾情况的学习⽅案、报考规划服务，以及后续学费由学⽣⾃⾏在⾃考官⽅平台缴纳的规则后，⽤⼾认可服务内容、明确确认缴纳1858元服务费⽤的对话瞬间，是促使⽤⼾完成服务签约的关键转折点。

• ⽂字描述例⼦：成交节点是指⽼师为⽤⼾讲解清楚专属学习计划、全程督学服务、分阶段备考辅导、历年真题精讲、全流程报考指导等1858元服务包含的全部权益，明确说明后续学费由学⽣⾃⾏在⾃考官⽹缴纳的规则后，⽤⼾确认签约服务、缴纳1858元服务费⽤的关键对话环节。

# • 聊天样本例⼦：样本1：

（3）什么是流失节点？

• 定义要求：指⽤⼾未完成服务签约的终⽌对话节点，具体表现为⽤⼾明确拒绝签约、超过48⼩时已读不回、明确表⽰选择其他机构，边界为⽤⼾发出拒绝话术，或最后⼀条有效对话后48⼩时⽆回复，即判定为流失节点。

• ⽂字描述例⼦：客⼾在得知1858元服务费⽤及服务模式后明确表⽰“太贵了不办了”，或⽼师发送完服务详情、专业政策介绍后，客⼾超过48⼩时未回复，或明确说“我去别的机构再看看”，均判定为流失节点。

• 聊天样本例⼦：样本1：

• ⽼师：我们的助学服务费⽤是1858元，包含了我刚才说的所有学习⽀持和全流程报考指导，后续的学费你直接在⾃考官⽹缴纳就可以。学⽣：这个服务费有点贵，我再考虑考虑吧。（后续48⼩时⽆任何回复）

（4）什么是⾼频节点？

定义要求：指被⼤量学⽣反复问及的共性问题、咨询流程或集中爆发的咨询阶段，是全量⽤⼾咨询中出现频次最⾼的对话内容。

• ⽂字描述例⼦：⾼频节点包括所有学⽣都会问的“⾃考报名⼊⼝在哪？”“毕业证多久能拿？”“准考证怎么打印？”“1858的服务费包含什么？”“学费在哪⾥交？”等通⽤问题，以及考前1个⽉集中爆发的备考相关问题。
